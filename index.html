What is Nakshora?

Nakshora is a utility-first open source CSS framework created to help developers build fast, beautiful, scalable, and expressive interfaces with one consistent design toolkit.

Utility First Architecture

Compose layouts and styles directly in HTML using practical utility classes for spacing, colors, typography, flexbox, grid, borders, shadows, positioning, sizing, and effects.

Multiple Aesthetic Systems

Build futuristic neon cyber interfaces, elegant pastel websites, clean minimalist products, organic nature-inspired layouts, or bold brutalist experiences inside the same framework.

Production Ready CSS Components

Nakshora includes buttons, cards, navbars, hero sections, forms, progress bars, alerts, modals, tabs, accordions, dropdowns, tooltips, skeleton loaders, and spinners.

Utility Classes in Nakshora

Nakshora is more than a component framework. It offers a rich utility class system for precise layout control and expressive styling.

Transform Utilities

How to Use Nakshora CSS Framework

Start using Nakshora instantly with the jsDelivr CDN and build modern UI with utility classes and components.

1. Add the Nakshora CDN

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/nakshora/nakshora@main/min.main.css">

2. Compose with Utility Classes

<div class="bg-neon-purple-500 text-mono-white p-6 rounded-xl shadow-xl">
  Hello Nakshora
</div>
